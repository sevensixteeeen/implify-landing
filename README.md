# Implify — Landing Page

> Stop reading. Start doing.

The official landing page for [Implify](https://implify-landing-phi.vercel.app) — a hands-on implementation platform where beginners stop consuming tutorials and start building real projects with AI guidance.

---

## What Is Implify?

Most learning platforms give you information. Implify gives you implementation.

You get dropped into a real project sandbox, given a mission, and an AI co-pilot guides you through it live — correcting mistakes, giving nudges, and making sure you finish with something actually working.

Not just coding. Frontend, backend, Git, data science, marketing, video editing, copywriting, design, and more.

## Features

- Waitlist form connected to Supabase
- RLS policy — emails stored securely (insert only)
- Fully responsive (mobile + desktop)
- No frameworks — pure HTML/CSS/JS
- Auto-deployed on Vercel on every push
